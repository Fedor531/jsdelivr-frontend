# Npm search
#### Описание: SPA для поиска npm-пакетов. 
#### Api: https://api-docs.npms.io/
#### Технологии:
* Vue 3
* Vuex
* Bootstrap 5
* Scss
#### Руководство
* Установка зависимостей
```
npm install
```
* Сборка проекта
```
npm run build
```
* Режим разработки
```
npm run serve
```
